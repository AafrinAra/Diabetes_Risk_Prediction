## 🔍 Overview

This project aims to predict whether a patient is likely to have diabetes based on health attributes such as:

- Pregnancies
- Insulin Level
- Age
- BMI
- Glucose
- Blood Pressure
- Diabetes Pedigree Function  

The dataset used in this project was taken from **Kaggle**, originally from the **National Institute of Diabetes and Digestive and Kidney Diseases**. The objective is to predict diabetes using diagnostic measurements. All patients are **females aged 21 and above** and belong to **Pima Indian heritage**.

A machine learning model (Random Forest classifier) is trained and deployed as a web application using **Django**.

---

## 🎯 Motivation

Diabetes is one of the fastest-growing health concerns globally. Early detection can prevent long-term health issues and save lives.

The motivation behind this project was:

- To build a complete **End-to-End ML pipeline**
- To explore deployment using Django
- To apply real-world machine learning workflows
- To work on a useful project that can help society
