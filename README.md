This project aims to predict whether a patient is likely to have diabetes based on health attributes such as:

Pregnancies  
Insulin Level  
Age  
BMI  
Glucose  
Blood Pressure  
Diabetes Pedigree Function

A Machine Learning based web application that predicts the risk of diabetes based on health parameters such as Glucose level, BMI, Age, Blood Pressure, etc.  
This project includes EDA, model training, and web deployment using Django.
